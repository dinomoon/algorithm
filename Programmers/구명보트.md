## 구명보트

### 탐욕법 - Level 2

### 문제 해설

정렬을 한 다음에 가장 무거운 사람과 가장 가벼운 사람을 계속 바꿔가며 구한다.

```js
function solution(people, limit) {
  var answer = 0;

  people.sort((a, b) => b - a);
  let l = 0;
  let r = people.length - 1;

  while (l < r) {
    let sum = people[l] + people[r];
    if (sum > limit) {
      l++;
    } else {
      l++;
      r--;
    }
    answer++;
  }

  if (l === r) answer++;
  return answer;
}
```
